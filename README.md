# Rajas Bhingarde Portfolio

An evidence-led portfolio for an early-career ML engineer applying AI to technical systems, forecasting, sensing, NLP, and operational automation.

## Product Principles

- Recruiter fit and strongest proof are visible before detailed narrative.
- Project claims identify ownership, constraints, evaluation, and evidence maturity.
- Essential content is semantic HTML; JavaScript only enhances interaction.
- The interface uses restrained engineering references without terminal or science-fiction styling.
- Motion communicates state and orientation rather than decorating the page.

## Interaction Model

- **Evidence Trace** lets visitors follow each project from input or context through engineering decisions to evidence.
- **Technical Review** uses native HTML disclosure so detailed project content remains keyboard-accessible and available without JavaScript.
- Primary navigation tracks the current content group and includes mobile Escape, outside-click, and resize handling.

## Technology

- Semantic HTML5
- CSS custom properties and responsive layouts
- Vanilla JavaScript progressive enhancement
- JSON-LD for person and featured-project metadata

No build step or package installation is required.

## Local Preview

Open `index.html` directly or run any static file server from the repository root.

## Content Sources

Recruiter artifacts are stored in `assets/docs/`. Project claims in the interface are limited to evidence already represented in the resume and accomplishments material.

## Accessibility

The site includes a skip link, visible focus states, native disclosure controls, keyboard-operable workflow traces, reduced-motion behavior, print styles, and content that remains readable without JavaScript.
